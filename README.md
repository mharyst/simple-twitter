# Simple Twitter app v1.0.1

Live version available at https://mharyst.github.io/simple-twitter/

## List of used technologies
- Webpack
- ReactJS
- Redux
- ES6 (with Babel)
- Bootstrap 4

---

## How to run app:

* Install dependencies:

```
npm i
```

* Start project:

  * OS Windows:

  ```
  npm start
  ```

  * On macOS or Linux use this command:

  ```
  npm run app-start
  ```

* Open in browser:

```
localhost:3030
```
